## @tools/cli

🍙 用于简化工程新建项目时的工具安装和配置，例如`eslint`,`git hooks`等，类似于[create-react-app](https://github.com/facebook/create-react-app)能为你免去[webpack](https://github.com/webpack/webpack)配置一样。

### 开始 🐠
安装到全局以使用工具
```bash
yarn
yarn build
yarn global add #全局安装
clis --version
```
### 用法 🚀
查看所有命令
```bash
clis -h
```
新建基于umi的项目
```bash
clis --umi <project-name>
```
